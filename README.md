# Factor

![factor](https://macplus.solutions/images/factor.png)

[Factor](https://macplus.solutions/docs/factor/) is a small Swift CLI tool built to solve a simple, concrete problem: finding the factors of a number. It began as a C++ utility I wrote to speed up coursework and was later reimplemented in Swift as a way to explore the platform and refine my approach. It emphasizes clarity over cleverness—simple structure, readable names, and predictable behavior—and serves as a clean foundation for building similar tools.

More broadly, it reflects a set of ideas I care about: simplicity over unnecessary abstraction, reducing toil through small, purposeful tools, and writing code for humans first. Naming, structure, and intent are treated as first-class concerns—not just for correctness, but for making the system easier to understand, extend, and trust.
