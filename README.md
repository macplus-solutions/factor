# Factor

![factor](https://macplus.solutions/images/factor.png)

A small Swift CLI for finding number factors—built as a clear, readable example of simple, intentional command-line tools.

### Usage
```
OVERVIEW: Print the factors of one or more integers.

USAGE: factor [<integers> ...]

ARGUMENTS:
  <integers>              The integers to factorize.

OPTIONS:
  --version               Show the version.
  -h, --help              Show help information.
```

[1, 2, 4, 8, 16, 32, 64, 128]
