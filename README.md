# Factor

![factor](https://macplus.solutions/images/factor.png)

[Factor](https://macplus.solutions/docs/factor/) is a small Swift CLI tool built to solve a simple, concrete problem: finding the factors of a number. It began as a utility I wrote to speed up coursework and was later reimplemented in Swift as a way to explore the platform. It emphasizes clarity over cleverness—simple structure, readable names, and predictable behavior—allowing it to serve as a clean foundation for building similar tools.

More broadly, it reflects a set of ideas I care about: simplicity over unnecessary abstraction, reducing toil through small, purposeful tools, and writing code for humans first. Repeatability is also a core concern—favoring patterns that can be reused and extended without friction. Naming, structure, and intent are treated as first-class concerns—not just for correctness, but for making systems easier to understand, extend, and trust.
